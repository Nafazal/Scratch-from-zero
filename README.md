# Scratch-from-zero

> My personal journey of learning programming from scratch.

## 📖 About

This repository is a collection of everything I learn while studying programming from the very beginning.

The purpose of this repository is to:

* Document my learning journey.
* Practice writing clean and understandable code.
* Track my progress over time.
* Build consistency through small projects and exercises.

This is **not** a collection of perfect code. Many files may contain mistakes, improvements, or multiple versions of the same solution. I intentionally keep them as a record of my learning process.

---

## 🎯 Learning Goals

* Learn programming fundamentals.
* Understand problem-solving using code.
* Build the habit of writing readable programs.
* Prepare for more advanced topics in computer science.
